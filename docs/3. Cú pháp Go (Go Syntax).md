# Cú pháp Go (Go Syntax)

Một tệp Go bao gồm các phần sau:

* Khai báo gói (Package declaration)
* Nhập các gói (Import packages)
* Các hàm (Functions)
* Các câu lệnh và biểu thức (Statements and expressions)

Hãy xem đoạn mã dưới đây để hiểu rõ hơn:

### Ví dụ

```go
package main

import ("fmt")

func main() {
  fmt.Println("Hello World!")
}

```

### Giải thích ví dụ

* **Dòng 1:** Trong Go, mọi chương trình đều là một phần của một gói (package). Chúng ta xác định điều này bằng từ khóa `package`. Trong ví dụ này, chương trình thuộc về gói `main`.
* **Dòng 2:** `import ("fmt")` cho phép chúng ta nhập các tệp nằm trong gói `fmt` (viết tắt của format, dùng để định dạng văn bản).
* **Dòng 3:** Một dòng trống. Go bỏ qua các khoảng trắng. Việc có các khoảng trắng trong mã giúp code dễ đọc hơn.
* **Dòng 4:** `func main() {}` là một hàm. Bất kỳ mã nào nằm trong dấu ngoặc nhọn `{}` của nó sẽ được thực thi.
* **Dòng 5:** `fmt.Println()` là một hàm có sẵn từ gói `fmt`. Nó được dùng để xuất/in văn bản. Trong ví dụ này, nó sẽ in ra "Hello World!".

**Lưu ý:** Trong Go, bất kỳ mã thực thi nào cũng phải thuộc về gói `main`.

---

## Câu lệnh trong Go (Go Statements)

`fmt.Println("Hello World!")` là một câu lệnh.

Trong Go, các câu lệnh được phân tách bằng cách kết thúc một dòng (nhấn phím Enter) hoặc bằng dấu chấm phẩy `;`.

Việc nhấn phím Enter sẽ tự động thêm dấu `;` vào cuối dòng một cách ngầm định (nó không hiển thị trong mã nguồn).

### Quy tắc về dấu ngoặc nhọn

Dấu ngoặc nhọn mở `{` **không thể** nằm ở đầu một dòng mới.

Hãy thử chạy đoạn mã sau (nó sẽ gây lỗi):

```go
package main
import ("fmt")

func main()  
{ // Lỗi: Dấu ngoặc nhọn không được để ở dòng mới như thế này
  fmt.Println("Hello World!")
}

```

## Mã nguồn rút gọn (Go Compact Code)

Bạn có thể viết mã gọn hơn như ví dụ dưới đây (tuy nhiên điều này không được khuyến khích vì nó làm cho mã khó đọc hơn):

### Ví dụ

```go
package main; import ("fmt"); func main() { fmt.Println("Hello World!");}

```

## Bài tập thực hành

Điền phần còn thiếu vào đoạn mã dưới đây để xuất ra dòng chữ "Hello World":

```go
package main
import ("fmt")

func main() {
  __________("Hello World!")
}

```

*(Đáp án: `fmt.Println`)*
