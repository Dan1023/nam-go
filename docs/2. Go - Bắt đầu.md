# Go - Bắt đầu (Go Getting Started)

Để bắt đầu sử dụng Go, bạn cần hai thứ:

1. **Một trình soạn thảo văn bản**, như VS Code, để viết mã Go.
2. **Một trình biên dịch**, như GCC, để dịch mã Go sang ngôn ngữ mà máy tính có thể hiểu được.

Có rất nhiều trình soạn thảo và trình biên dịch để lựa chọn. Trong bài hướng dẫn này, chúng ta sẽ sử dụng một IDE (xem bên dưới).

### Cài đặt Go

Bạn có thể tìm các tệp cài đặt liên quan tại [https://golang.org/dl/](https://golang.org/dl/).

Hãy làm theo các hướng dẫn tương ứng với hệ điều hành của bạn. Để kiểm tra xem Go đã được cài đặt thành công hay chưa, bạn có thể chạy lệnh sau trong cửa sổ terminal (cửa sổ lệnh):

```bash
go version

```

Lệnh này sẽ hiển thị phiên bản Go mà bạn đã cài đặt.

### Cài đặt IDE cho Go

Một IDE (Môi trường phát triển tích hợp) được sử dụng để vừa chỉnh sửa VÀ vừa biên dịch mã.

Các IDE phổ biến bao gồm **Visual Studio Code (VS Code)**, **Vim**, **Eclipse**, và **Notepad**. Tất cả đều miễn phí và có thể được sử dụng để chỉnh sửa cũng như gỡ lỗi (debug) mã Go.

*Lưu ý: Các IDE dựa trên web cũng có thể hoạt động, nhưng chức năng sẽ bị hạn chế.*

Chúng tôi sẽ sử dụng **VS Code** trong bài hướng dẫn này, vì đây là một điểm bắt đầu tốt. Bạn có thể tìm phiên bản VS Code mới nhất tại [https://code.visualstudio.com/](https://code.visualstudio.com/).

### Khởi đầu nhanh với Go (Go Quickstart)

Hãy cùng tạo chương trình Go đầu tiên của bạn.

1. Khởi chạy trình soạn thảo **VS Code**.
2. Mở trình quản lý tiện ích mở rộng (extension manager) hoặc nhấn tổ hợp phím `Ctrl + Shift + x`.
3. Trong ô tìm kiếm, gõ **"go"** và nhấn Enter.
4. Tìm tiện ích mở rộng Go của nhóm **Go team at Google** và nhấn **Install** để cài đặt.
5. Sau khi cài đặt xong, mở bảng lệnh (command palette) bằng cách nhấn `Ctrl + Shift + p`.
6. Chạy lệnh `Go: Install/Update Tools`.
7. Chọn tất cả các công cụ được cung cấp và nhấn **OK**.

Bây giờ VS Code đã được cấu hình để sử dụng Go.

Mở cửa sổ terminal và gõ:

```bash
go mod init example.com/hello

```

*Đừng lo lắng nếu bạn chưa hiểu tại sao phải gõ lệnh trên. Hãy coi đó là việc luôn phải làm, và bạn sẽ tìm hiểu kỹ hơn ở các chương sau.*

Tạo một tệp mới (**File > New File**). Sao chép và dán đoạn mã sau, sau đó lưu tệp với tên `helloworld.go` (**File > Save As**):

**helloworld.go**

```go
package main

import ("fmt")

func main() {
  fmt.Println("Hello World!")
}

```

Bây giờ, hãy chạy đoạn mã: Mở terminal trong VS Code và gõ:

```bash
go run .\helloworld.go

```

**Kết quả:**
`Hello World!`

Chúc mừng! Bạn đã vừa viết và thực thi thành công chương trình Go đầu tiên của mình.

Nếu bạn muốn lưu chương trình dưới dạng một tệp thực thi (executable), hãy gõ và chạy lệnh:

```bash
go build .\helloworld.go

```

### Học Go tại W3Schools

Khi học Go tại W3Schools.com, bạn có thể sử dụng công cụ "Try it Yourself" (Tự mình thử nghiệm). Nó sẽ hiển thị cả mã nguồn và kết quả, giúp bạn dễ dàng hiểu từng phần khi chúng ta tiến xa hơn.

**Ví dụ:**

* **Mã nguồn:**
```go
package main
import ("fmt")
func main() {
  fmt.Println("Hello World!")
}

```


* **Kết quả:** `Hello World!`
